# images/

Place all site images in this folder and reference them in index.html with:

  <img src="images/your-file.jpg" alt="Description">

## Suggested images for this page

| Filename              | Where used                        | Recommended size  |
|-----------------------|-----------------------------------|-------------------|
| `headshot.jpg`        | Hero section — founder portrait   | 600 × 800 px      |
| `og-image.jpg`        | Open Graph social preview         | 1200 × 630 px     |
| `favicon.ico`         | Browser tab icon                  | 32 × 32 px        |
| `logo.svg`            | Nav logo (optional graphic mark)  | vector            |

## Adding a favicon

Add this inside <head> in index.html:

  <link rel="icon" href="images/favicon.ico">

## Adding Open Graph metadata

Add inside <head> in index.html:

  <meta property="og:title"       content="Reimagining Consulting">
  <meta property="og:description" content="Enterprise-grade consulting for financial services and utilities.">
  <meta property="og:image"       content="images/og-image.jpg">
  <meta property="og:type"        content="website">
